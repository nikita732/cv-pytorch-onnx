# Отчет по квантизации (Задание 8)

## Результаты измерений
- **Accuracy FP32**: 96.80%
- **Accuracy INT8**: 10.60%
- **Разница (Drop)**: 86.20%

## Анализ
При использовании динамической квантизации точность упала до уровня случайного угадывания (~10%). 
Это связано с тем, что веса EfficientNet чувствительны к разбросу значений, и простое приведение к INT8 без калибровки (Static Quantization) разрушило предсказательную способность модели.

## Вывод
Для данной модели динамическая квантизация неприменима. Необходимо использовать статическую квантизацию с калибровочным датасетом для сохранения точности.
